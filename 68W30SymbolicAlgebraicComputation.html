68W30 Symbolic computation and algebraic computation

This map presents the organization of the references and resources for symbolic computation and algebraic computation. It contains items that pertain also to 68N15 (Programming languages).

The hyperlink in this node (or at the top of the web page) points to the local directory where the files for this topic are stored.

The codes "68N15" and "68W30" correspond to the MSC2010 classification for:

68 Computer science. Under this topic 68N is for software, 68W for algorithms.

68N15 corresponds to programming languages.

68W30 corresponds to symbolic computation and algebraic computation.
