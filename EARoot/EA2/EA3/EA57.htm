The Basic Path Basic Path
Details:
Steps:
1. El Usuario abre  la página inicial (frmInicioBiblioteca)
2. El Usuario solicita realizar una consulta
3. El sistema de Biblioteca abre la página para consulta de ficha bibliográfica (frmPagInicioConsulta)
4. El usuario escribe el primer apellido del autor
5. El usuario elecciona "Autor" como criterio de búsqueda
6. El usuario solicita iniciar la búsqueda
7. El sistema responde con la lista de autores a los que corresponde el apellido(frmListaAutores)
8. El usuario selecciona el autor
9. El sistema despliega la lista obras (frmListaObras)
10. El usuario selecciona la obra.
11. El sistema presenta la ficha bilbiográfica (frmFicha)
12. El caso de uso termina cuando el usuario solicita salir del sistema
Exceptions:
4a. Consulta por título (at 9)
4b. Consulta por ISBN (at 11)
