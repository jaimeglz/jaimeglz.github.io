Scenario Type
Secuencia para realizar una consulta Basic Path
Details:
Este caso de uso inicia cuando el usuario accede a la página web de la biblioteca. <br /></p><p>El usuario solicita iniciar la consulta de la ficha bibliográfica.<br /></p><p>El sistema de biblioteca despliega la página para consulta de ficha bibliográfica.<br /></p><p>El usuario: <br /></p><p><ul><li>escribe el primer apellido del autor, el título o el ISBN en el campo para texto;</li><br /></p><p><li>selecciona "Autor", "Título" o "ISBN" como criterio de búsqueda </li><br /></p><p><li>solicita iniciar la búsqueda (consúltese el ejemplo)</li><br /></p><p></ul><br /></p><p><b>En la consulta por autor</b><br /></p><p>El sistema responde con una lista de los autores a los que corresponde el apellido.<br /></p><p>El usuario selecciona el autor.<br /></p><p><br /></p><p><b>En la consulta por título, o después de seleccionar autor</b><br /></p><p>El sistema presenta una lista de las obras que corresponden al autor o al título selecionado.<br /></p><p>El usuario selecciona la obra.<br /></p><p><br /></p><p><b>En la consulta por ISBN, o después de seleccionar la obra</b><br /></p><p>El sistema presenta la ficha bilbiográfica. (Consúltese el ejemplo contenido en el modelo de requisitos, sección interaz de usuario.)<br /></p><p>El caso de uso termina cuando el usuario abandona la página que contiene la ficha bibliográfica.<br /></p>
Steps:
1. El Usuario abre  la página inicial (frmInicioBiblioteca)
2. El Usuario solicita realizar una consulta
3. El sistema de Biblioteca abre la página para consulta de ficha bibliográfica (frmPagInicioConsulta)
4. El usuario escribe el primer apellido del autor
5. El usuario elecciona "Autor" como criterio de búsqueda
6. El usuario solicita iniciar la búsqueda
7. El sistema responde con la lista de autores a los que corresponde el apellido(frmListaAutores)
8. El usuario selecciona el autor
9. El sistema despliega la lista obras (frmListaObras)
10. El usuario selecciona la obra.
11. El sistema presenta la ficha bilbiográfica (frmFicha)
12. El caso de uso termina cuando el usuario solicita salir del sistema
Exceptions:
4a. Consulta por título (at 9)
4b. Consulta por ISBN (at 11)
Nueva consulta Alternate
Details:
Steps:
1. El usuario selecciona la opción para realizar otra consulta
Consulta por ISBN Alternate
Details:
Steps:
1. El usuario escribe el ISBN en el campo de frmPagInicioConsulta para texto
2. El usuario selecciona "ISBN" como criterio de búsqueda
3. Solicita iniciar la búsqueda de la obra específica
Consulta por título Alternate
Details:
Steps:
1. El usuario escribe el título en el campo para texto
2. El usuario selecciona "Título" como criterio de búsqueda
3. El usuario solicita iniciar la búsqueda
