Secuencia para realizar una consulta<br /></p><p>Este caso de uso inicia cuando el usuario accede a la página web de la biblioteca. <br /></p><p>El usuario solicita iniciar la consulta de la ficha bibliográfica.<br /></p><p>El sistema de biblioteca despliega la página para consulta de ficha bibliográfica.<br /></p><p>El usuario: <br /></p><p><ul><li>escribe el primer apellido del autor, el título o el ISBN en el campo para texto;</li><br /></p><p><li>selecciona "Autor", "Título" o "ISBN" como criterio de búsqueda </li><br /></p><p><li>solicita iniciar la búsqueda (consúltese el ejemplo)</li><br /></p><p></ul><br /></p><p><b>En la consulta por autor</b><br /></p><p>El sistema responde con una lista de los autores a los que corresponde el apellido.<br /></p><p>El usuario selecciona el autor.<br /></p><p><br /></p><p><b>En la consulta por título, o después de seleccionar autor</b><br /></p><p>El sistema presenta una lista de las obras que corresponden al autor o al título selecionado.<br /></p><p>El usuario selecciona la obra.<br /></p><p><br /></p><p><b>En la consulta por ISBN, o después de seleccionar la obra</b><br /></p><p>El sistema presenta la ficha bilbiográfica. (Consúltese el ejemplo contenido en el modelo de requisitos, sección interaz de usuario.)<br /></p><p>El caso de uso termina cuando el usuario abandona la página que contiene la ficha bibliográfica.<br /></p>
